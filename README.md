# 🎮 Tetris - JavaFX 테트리스 게임

> 2025 NL 소프트웨어공학 Team 12

## 📖 프로젝트 소개

JavaFX 기반의 테트리스 게임입니다. 클래식 테트리스의 재미를 현대적인 UI와 다양한 게임 모드로 즐길 수 있습니다.

## ✨ 주요 기능

### 🎯 게임 모드
- **싱글 플레이**: 클래식 테트리스 모드
- **아이템 모드**: 특수 아이템 블록이 등장하는 모드
- **타임어택 모드**
- **로컬 멀티플레이**: 한 컴퓨터에서 2인 대전
- **P2P 멀티플레이**: 네트워크를 통한 1:1 대전

### 🏆 추가 기능
- 고스트 피스 표시
- 피스 홀드 기능
- 네트워크 Robust한 멀티플레이

## 📁 프로젝트 구조

```
src/main/java/org/example/
├── App.java                 # 애플리케이션 진입점
├── controller/              # MVC 컨트롤러
├── model/                   # 게임 로직 및 데이터 모델
├── service/                 # 서비스 레이어 (설정, 색상, 디스플레이 관리)
└── view/                    # UI 뷰 컴포넌트
    └── component/           # 재사용 가능한 UI 컴포넌트

src/main/res/
├── icons/                   # 아이콘 리소스
└── styles/
    └── tetris.css           # 게임 스타일시트
```
